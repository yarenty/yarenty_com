This post announces version 0.5.0 of Kowalski, a modular AI agent framework built in Rust. The update includes significant refactoring, a new multi-agent orchestration layer, an improved CLI, and enhanced pluggable tools for tasks like code analysis and web scraping. It emphasizes the framework's full-stack Rust nature (no Python dependencies) and encourages community contributions.
